Pitch

Hi! I would like to take the opportunity offered by this proect to present my final project: DesignDetective! the AI design assitant for everyone. Simply install the web extension for your favorite browser, (work on chromium browser, not tested on firefox ones.), and get you are all set ! DesignDetective uses Tensorflow pre-trained AI model to analyse the present web page, (using her url), and return the font used, the hexadecimals value of the color palette & where to find them, or close-enough free alternatives. DesignDetective will be free and opensource and will be maintained through my Github. DesignDetective will not collect any users data for now, since i don't plan on monetize the project. Users are limited to 10 API calls a day. (Otherwise and i would have to pay per API calls.) which should be plenty for your design needs ! Get the extension all of the designers form the web: streamers, content creator, web designers, graphist... Let DesignDetective do the hard work for you! (planned release on the 25th of December. Work in proress.)
