The Pitch

Hi! I would like to take the opportunity offered by this project to present my final one: DesignDetective! the AI design assitant for everyone. Simply install the web extension for your favorite browser, (work on chromium browser, not tested on firefox ones.), and get you are all set ! DesignDetective uses Tensorflow pre-trained AI model to analyse the present web page, (using her url), and return the font used, the hexadecimals value of the color palette & where to find them, or close-enough free alternatives. DesignDetective will also be free and opensource and will be maintained through my Github. DesignDetective will not collect any users data for now, since i don't plan on monetize the project. Users are limited to 10 API calls a day. (Otherwise and i would have to pay per API calls.) which should be plenty for your design needs ! Get the extension all of the designers from the web: streamers, content creator, web designers, graphist... Let DesignDetective do the hard work for you! (planned release on the 25th of December. Work in proress.)

Let's get technical !

Issue is: For now, i don't have any server to deploy my web extension, nor to train an AI model. I can't monetize my project for now because i can't assure my customer will have a good support, seemless experience nor secure account. I am still attending CS50x classes, i could figure this out later on. For now, no account, no servers, no data: You install the extension, you analyse website. Easy as that. I am planning on using TensorFlow trained AI for tis task: Easy, Light-weight (run in their own servers, with an API call ID), Basic "import" in JavaScript. Might change my minds in the future, because the free tier of TensorFlow is 100 API calls a day / users. Which seems insanly good, but some, more curious users, could push that boundary. After 100 API calls, i have to pay per API calls (which is no good.)
